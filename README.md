# 42DiskUsage

## 터미널에 붙여넣으세요
``` shell
git clone git@github.com:JaeJunday/42DiskUsage
cd 42DiskUsage
make re
```
## 사용방법
``` shell
cache
```

## 오류정보
- oh-my-zsh을 사용하는 경우 zshrc에 스크립트삽입하는 행위를 위험한 동작으로 간주해서 제대로 동작하지 않을 수 있습니다.
